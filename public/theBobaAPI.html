The Boba API

The Boba API provides information on local and international bubble tea franchises, including their locations, menus, and additional details.

Endpoints

/franchises

Returns a list of all the franchises with information available in this API in the format "display name:short name".

/franchiseInfo

Returns a JSON file including the information of every franchise available in this API.

/franchises/:name

Given a valid franchise name, returns a JSON file including the information of the specified franchise.

Disclaimer

The franchises in The Boba API are included on the criteria that the creator has visited them. Last updated November 2019.
